# Himed 제품 카탈로그 분석 및 웹사이트 정보 아키텍처

## 제품 분류 체계 (Product Taxonomy)

### 1. MATRIX® System (표면 처리 및 코팅)
**용도**: 티타늄 임플란트 표면 처리 및 코팅

- **MCD Apatitic Abrasive** (MCD)
  - 입자 크기: 6개 옵션 (MCD180, MCD160, MCD140, MCD120, MCD100, MCD20)
  - 페이지: 2-4

---

### 2. Hydroxyapatite (HA) 제품군
**용도**: 골 이식재, 골 시멘트, 크로마토그래피, 임플란트 코팅

#### 2.1 분말 형태 (Powder Forms)
- **MCHA** - Hydroxyapatite (MC-Type) | Solid-State Sintered
  - 입자 크기: 2개 옵션 (MCHA15, MCHA10)
  - 페이지: 5-7
  - 특징: 플라즈마 스프레이 코팅 전용

- **SWHA** - Hydroxyapatite | Precipitated + Sintered
  - 입자 크기: 6개 옵션 (SWHA60, SWHA50, SWHA30, SWHA25, SWHA20, SWHA13)
  - 페이지: 8-10

- **UWHA** - Hydroxyapatite | Precipitated + Unsintered
  - 입자 크기: 4개 옵션 (UWHA50, UWHA30, UWHA20, UWHA13)
  - 페이지: 11-13
  - 특징: 소결 전 분말, 다공성 구조 제작 가능

- **HAWHISK-S** - Hydroxyapatite Whiskers
  - 형태: 로드 모양 단결정 입자 (1개 옵션)
  - 페이지: 14-16
  - 특징: 높은 종횡비 (3:1 ~ 10:1)

#### 2.2 디스크 형태 (실험실용 기판, Disc Forms)
- **CAD** - Calcium Deficient HA Discs | Unsintered
  - 크기: 4개 옵션 (CAD35, CAD30, CAD25, CAD20)
  - 페이지: 17-19
  - 특징: 세포/박테리아 배양, 스퍼터링 타겟

- **HAD** - Dense HA Discs | Sintered
  - 크기: 5개 옵션 (HAD70, HAD65, HAD50, HAD40, HAD15)
  - 페이지: 20-22

- **HADEL** - Enamel-like HA Discs | Sintered
  - 크기: 5개 옵션 (HADEL02, HADEL03, HADEL04, HADEL22, HADEL30)
  - 페이지: 23-25
  - 특징: 반투명, 치아 법랑질 유사

---

### 3. Tricalcium Phosphate (TCP) 제품군
**용도**: 골 이식재, 조직공학 스캐폴드, 골 시멘트

- **ATCP** - Alpha-tricalcium phosphate
  - 입자 크기: 2개 옵션 (ATCP20, ATCP13)
  - 페이지: 26-28
  - 용해도: HA, b-TCP보다 높음

- **SSBTCP** - Beta-TCP | Solid-State Sintered
  - 입자 크기: 7개 옵션 (SSBTCP30, SSBTCP24, SSBTCP15, SSBTCP20, SSBTCP43, SSBTCP13, SSBTCP14)
  - 페이지: 29-31

- **SWBTCP** - Beta-TCP | Precipitated + Sintered
  - 입자 크기: 7개 옵션 (SWBTCP60, SWBTCP50, SWBTCP30, SWBTCP25, SWBTCP20, SWBTCP13, SWBTCP14)
  - 페이지: 32-34
  - 특징: 침전법, 더 높은 활성도

- **UTCP** - Tricalcium Phosphate | Precipitated + Unsintered
  - 입자 크기: 5개 옵션 (UTCP50, UTCP30, UTCP20, UTCP13, UTCP14)
  - 페이지: 35-37
  - 특징: 소결 시 b-TCP로 전환

---

### 4. Tetracalcium Phosphate (TTCP)
**용도**: 골 시멘트, 치아 재광화

- **TTCP** - Tetracalcium phosphate
  - 입자 크기: 2개 옵션 (TTCP20, TTCP13)
  - 페이지: 38-40
  - 용해도: 매우 높음 (ACP 다음)

---

### 5. Biphasic Calcium Phosphate (BCP)
**용도**: 골 이식재, 조직공학 스캐폴드, CaP-폴리머 복합 임플란트

#### 5.1 소결형 (Sintered)
- **S1BCP** - Biphasic Calcium Phosphate | Sintered
  - HA/b-TCP 비율: 5개 옵션
    - 60/40 (S1BCP60/40)
    - 50/50 (S1BCP50/50)
    - 40/60 (S1BCP40/60)
    - 30/70 (S1BCP30/70)
    - 20/80 (S1BCP20/80)
  - 각 비율당 입자 크기: 4개 옵션 (35, 03, 20, 13)
  - 총 SKU: 20개
  - 페이지: 41-43
  - 특징: HA ↑ = 안정성 ↑, b-TCP ↑ = 생체활성/흡수율 ↑

#### 5.2 비소결형 (Unsintered)
- **U1BCP** - Biphasic Calcium Phosphate | Unsintered
  - HA/b-TCP 비율: 5개 옵션
    - 85/15 (U1BCP85/15)
    - 65/35 (U1BCP65/35)
    - 55/45 (U1BCP55/45)
    - 25/75 (U1BCP25/75)
    - 15/85 (U1BCP15/85)
  - 각 비율당 입자 크기: 4개 옵션 (35, 03, 20, 13)
  - 총 SKU: 20개
  - 페이지: 44-46

## 제품 페이지 레이아웃 제안

각 제품 페이지는 다음 섹션으로 구성:

### 1. Hero Section
- 제품명 + 약어
- 주요 특징 (1-2 문장)
- CTA: Download PDF, Request Quote

### 2. Tab Navigation
- **Overview** (개요)
- **Specifications** (상세 스펙)
- **Applications** (응용 분야)
- **Related Products** (관련 제품)

### 3. Overview Tab
- 제품 설명
- SEM 이미지
- 주요 장점 (bullet points)
- 제조 공정 설명

### 4. Specifications Tab
- **Physical Properties** (테이블)
  - Appearance
  - Density
  - Particle size
  - etc.

- **Chemical Properties** (테이블)
  - Chemical formula
  - Composition (ICP, XRD)
  - Trace elements

- **Particle Sizes** (드롭다운 선택기)
  - Catalog #
  - Size range
  - Acceptable range

### 5. Applications Tab
- 주요 응용 분야
- 관련 프로세스
- 사용 사례

### 6. Related Products Tab
- 같은 카테고리 제품
- 대체 가능 제품
- 보완 제품

## 기술적 구현 방향

### 데이터 구조
- 제품 정보를 JSON 또는 YAML로 구조화
- 각 제품은 고유 ID, 카테고리, 태그, 스펙 포함
- 관계형 데이터로 연관 제품 링크

### 페이지 생성
- Cloudflare Pages + 정적 사이트 생성기 (11ty, Astro, Next.js 등)
- 제품 데이터로부터 자동으로 페이지 생성
- 빠른 로딩을 위한 최적화

### 디자인 시스템
- 임상/과학적 톤
- 테이블 중심 레이아웃
- 명확한 타이포그래피 계층
- 최소한의 색상 사용 (흰색/회색 배경)
- 데이터시트 스타일 테이블
